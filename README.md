# obsidian-vault

This repository contains my Obsidian vault configuration, templates, and helper scripts used to create and manage notes.

## Contents

- templates/: note templates (e.g. Zettelkasten Note)
- templates/Scripts/: helper scripts used by templates (e.g. note_id.js)

## Setup

1. Clone the repository to your local machine:
```bash
git clone https://github.com/ErlanRG/obsidian-vault.git ~/Documents/Notes
```
2. Open Obsidian and add the cloned folder as a vault.
3. Ensure all the plugins used in the templates are installed and enabled in Obsidian.
4. Customize the templates and scripts as needed to fit your workflow.
